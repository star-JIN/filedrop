      GitHub 설정

      GitHub Personal Access Token과 저장소 정보를 입력하세요.
      브라우저 로컬에만 저장되며 외부 전송되지 않습니다.

      ✅ 저장됨
      처음 설정하기
      1. GitHub에서 새 저장소를 만드세요 (Private 추천)
      2. Personal Access Token 생성 → repo 권한 체크
      3. 위 설정에 토큰, 사용자명, 저장소명 입력 후 저장
      4. GitHub 탭에서 파일 업로드/다운로드!
      P2P 연결 정보

      P2P 모드는 WebRTC를 사용합니다. 시그널링에 PeerJS 공개 서버를 사용하며, 파일 데이터는 기기 간 직접 전송됩니다. 회사 방화벽이 WebRTC를 차단하는 경우 GitHub 모드를 사용하세요.
