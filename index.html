처음 설정하기
  1. GitHub에서 새 저장소를 만드세요 (Private 추천) — github.com/new
  2. Personal Access Token을 생성하고 repo 권한을 체크하세요 — settings/tokens/new
  3. 위 설정에 토큰, 사용자명, 저장소명을 입력한 뒤 저장하세요.
  4. 📦 GitHub 탭에서 파일을 업로드/다운로드하세요!
